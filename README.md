# TwitterApp

-Instalar y crear un nuevo projecto base usando la respectiva documentacion de flask: http://flask.palletsprojects.com/en/1.1.x/

-Instalar la libreria que consumira los datos de la API de Twitter "pip install tweepy"

-Obtener un token de desarrollador "apps.twitter.com"

-Usar los "alias" que vienen por cada usuario en la busqueda interna de la red social, ejemplo: "@example" excluyendo el simbolo "@"
